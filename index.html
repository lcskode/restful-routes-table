RESTful Routes

A table of all 7 RESTful routes

Name Path HTTP Verb Purpose Mongoose Method
Index /blogs GET List all blogs Blog.find()
New /blogs/new GET Show new blogs form N/A
Create /blogs POST Create new blog, then redirect somewhere Blog.create()
Show /blogs/:id GET Show info about one specific blog Blog.findById()
Edit /blogs/:id/edit GET Show edit form for one blog Blog.findById()
Update /blogs/:id PUT Update a particuler blog, then redirect somewhere Blog.findByIdAndUpdate()
Destroy /blogs/:id DELETE Delete a particuler blog, then redirect somewhere Blog.findByIdAndRemove()
